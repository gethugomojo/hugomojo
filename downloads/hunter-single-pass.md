# HugoMojo Hunter Single Pass

## Role Signal

The Hunter is built for fast execution, client work, and direct service monetization.

## First 7 Days

1. Select one offer: AI writing, AI automation setup, or AI visual asset delivery.
2. Build a one-page service menu with three clear deliverables.
3. Create 10 prospect messages using the outreach prompt.
4. Send 20 targeted messages.
5. Deliver one sample asset to prove speed.
6. Ask for one paid trial project.
7. Document the delivery steps into a repeatable checklist.

## Starter Prompt

```text
Act as an AI service operator. Help me turn this skill into a clear client offer. My skill is [SKILL]. My target customer is [CUSTOMER]. Create three deliverables, a simple pricing ladder, and a 7-day outreach plan.
```

## Delivery Checklist

- Clear offer
- Clear customer type
- Clear delivery timeline
- Clear revision rule
- Clear price
- Clear next step
